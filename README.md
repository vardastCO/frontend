This is a [Next.js](https://nextjs.org/) project.

## Getting Started

Copy ENV file

```bash
cp .env.example .env
```

Install dependencies

```bash
pnpm install
```

Generate GraphQL types

```bash
pnpm codgen
```

Build app

```bash
pnpm build
```
